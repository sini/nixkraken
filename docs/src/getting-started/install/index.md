# Installation

Although the recommended method to install NixKraken is through [Flakes][nix-wiki-flakes], we also fully support installing it on [channel-based setups][nixos-manual-channels].

Additionally, NixKraken is compatible both with standalone [Home Manager][hm] configurations and NixOS integrated ones.

Please refer to the installation guides below to learn how to install NixKraken and its complementary packages.

- [NixKraken without Flakes][doc-nonflakes] \
